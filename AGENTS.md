# Project Instructions

## Protected Favicon

Hard rule: the favicon must never be touched, modified, renamed, replaced, regenerated, reformatted, optimized, deleted, or removed unless Tresor explicitly asks for a favicon change.

Protected items:
- `public/favicon.svg`
- The favicon link in `index.html`: `<link rel="icon" type="image/svg+xml" href="/favicon.svg" />`

Before any broad cleanup, asset change, build-output change, or `index.html` edit, verify that the favicon file and favicon link are still intact.
